
Module: Studizz Bot
Author: Johann Liberman <johann.liberman@studizz.fr>

Description
===========
Adds the Studizz Bot system to your website.

Requirements
============

* Studizz account

Installation
============
Copy the 'studizzbot' module directory in to your Drupal
sites/all/modules directory as usual.

Usage
=====
In the settings page enter your Studizz account number.

All pages will now have the required JavaScript added to the
HTML footer can confirm this by viewing the page source from
your browser.
